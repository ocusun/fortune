# 「오늘의 나」 시스템 설계서

## 1. 8유형 정의

각 유형은 한국 자연물에 빗대어 표현. 단정적 성격 분류가 아닌 **오늘의 결**로 제시.

### 1) 깊은 호수 (深湖)
- **결**: 사색·내면·고요
- **오늘의 당신**: 말보다 생각이 앞서는 하루. 표면은 잔잔하지만 안에서는 많은 것이 일어나고 있습니다.
- **어울리는 것**: 혼자만의 시간, 깊이 있는 책, 잔잔한 음악, 글쓰기
- **필요한 것**: 답을 서두르지 않고 머무는 시간

### 2) 흐르는 강 (流江)
- **결**: 균형·적응·중용
- **오늘의 당신**: 어떤 자리에서도 결을 맞출 수 있는 하루. 강한 주장보다 조화를 택하는 마음.
- **어울리는 것**: 사람들과의 가벼운 만남, 익숙한 산책로, 따뜻한 음식
- **필요한 것**: 너무 많은 사람의 결을 맞추다 지치지 않도록 자기 자리도 챙기기

### 3) 타오르는 불 (烈火)
- **결**: 열정·추진·창조
- **오늘의 당신**: 안에서 무언가가 끓고 있는 하루. 미뤄둔 일에 손을 대기 좋은 때.
- **어울리는 것**: 도전하는 영화, 추진력 있는 인물 전기, 야외 활동
- **필요한 것**: 뜨거움을 다스리는 한 박자의 쉼

### 4) 바람 부는 들 (風野)
- **결**: 자유·변화·새로움
- **오늘의 당신**: 한 자리에 머무르지 못하는 하루. 새로운 풍경이 필요한 마음.
- **어울리는 것**: 여행기, 로드무비, 안 가본 동네, 새로운 음식
- **필요한 것**: 변화 속에서도 흔들리지 않을 작은 중심점 하나

### 5) 단단한 바위 (盤石)
- **결**: 안정·신뢰·꾸준함
- **오늘의 당신**: 흔들림 없이 자기 자리를 지키는 하루. 천천히, 그러나 분명하게.
- **어울리는 것**: 묵직한 고전, 가족 드라마, 익숙한 공간
- **필요한 것**: 단단함 속에 가끔은 숨 쉴 틈도 두기

### 6) 빛나는 새벽 (黎明)
- **결**: 희망·시작·돌파
- **오늘의 당신**: 어둠을 막 지나온 하루. 작은 빛 하나가 멀리 보입니다.
- **어울리는 것**: 성장 드라마, 첫 시작에 관한 책, 일찍 일어나기
- **필요한 것**: 새벽이 짧으니 그 빛이 사라지기 전 한 걸음 내딛기

### 7) 무르익은 가을 (秋熟)
- **결**: 성숙·관조·정리
- **오늘의 당신**: 한 계절이 마무리되는 하루. 거두고 정리하기 좋은 마음.
- **어울리는 것**: 회고록, 인생 영화, 묵은 사진 정리, 깊은 차 한 잔
- **필요한 것**: 정리는 끝이 아니라 다음을 위한 자리 만들기

### 8) 깨어나는 봄 (春芽)
- **결**: 재생·회복·시작
- **오늘의 당신**: 오랜 추위가 끝나가는 하루. 작은 새싹 하나가 안에서 올라옵니다.
- **어울리는 것**: 회복에 관한 이야기, 따뜻한 가족 드라마, 산책
- **필요한 것**: 새싹은 천천히 자라니 너무 빨리 결과를 바라지 말기

## 3. 유형 매핑 로직

각 응답 옵션마다 8유형에 대한 점수가 부여됨. 모든 응답의 점수를 합산해 가장 높은 유형 1개 + 보조 유형 1개를 반환.

### 점수 부여 예시 (Q1: 지금 기분)

| 응답 | 깊호 | 흐강 | 타불 | 바들 | 단바 | 빛새 | 무가 | 깨봄 |
|---|---|---|---|---|---|---|---|---|
| 절망 | +3 | 0 | 0 | 0 | +1 | 0 | +2 | +2 |
| 우울 | +3 | +1 | 0 | 0 | +1 | 0 | +2 | +1 |
| 평범 | +1 | +3 | 0 | +1 | +3 | 0 | +1 | 0 |
| 기쁨 | 0 | +1 | +2 | +2 | +1 | +3 | 0 | +2 |
| 환희 | 0 | 0 | +3 | +2 | 0 | +3 | 0 | +1 |

(전체 매핑표는 별도 JSON 파일로 관리)


## 4. 365일 위인·명언 DB 구조

```json
{
  "5-2": {
    "date": "5월 2일",
    "figure": "레오나르도 다 빈치",
    "country": "이탈리아",
    "context": "1519년 별세",
    "quote": "경험은 결코 잘못되지 않는다. 잘못된 것은 우리의 판단이다.",
    "quote_original": "Experience does not err...",
    "tags": ["관조", "성숙", "직관"],
    "linked_types": ["무르익은 가을", "깊은 호수"]
  }
}
```

각 항목에 `linked_types`를 추가해 그날의 위인이 어떤 유형과 어울리는지 표시. 진단 결과와 그날의 명언이 연결되는 경우 "오늘은 당신의 결과 잘 맞는 위인의 명언이 함께합니다" 같은 부가 메시지 가능.


## 5. 책 100 / 영화 100 매핑 구조

```json
{
  "book_id": "B042",
  "title": "어린 왕자",
  "author": "생텍쥐페리",
  "year": 1943,
  "primary_types": ["깊은 호수", "깨어나는 봄"],
  "secondary_types": ["무르익은 가을"],
  "mood_fit": ["우울", "평범"],
  "keywords": ["관계", "본질", "어린 시절"],
  "recommend_reason_template": [
    "오늘처럼 마음이 잠잠한 날, 어린 왕자의 한 문장이 멀리 있는 별처럼 다가올지 모릅니다",
    "사람과 사람 사이의 본질을 다시 묻고 싶을 때 펼치기 좋은 책",
    "여우의 길들임에 관한 이야기가 오늘의 당신께 닿을 수 있습니다"
  ]
}
```

영화도 동일한 구조. `recommend_reason_template`이 3~5개씩 있어서 같은 책이 추천되어도 표현이 다르게 나옴.


## 6. AI 개인화 풀 (368개)

8유형 × 46개 변주 = 368개의 "오늘의 처방"

```json
{
  "type": "깊은 호수",
  "prescriptions": [
    "오늘의 당신께 — 답을 정하지 마세요. 호수는 흐트러진 후에 더 깊어집니다.",
    "오늘의 당신께 — 누군가에게 말하지 않아도 되는 일이 있습니다. 그 침묵을 지켜주세요.",
    "오늘의 당신께 — 깊이 들어갈수록 차가워지지만, 그 안에 답이 있습니다.",
    ... (46개)
  ]
}
```

응답 시 사용자의 응답 시드(예: `Q1=2,Q2=4,Q3=1,...` → 해시값)로 46개 중 하나를 선택. 같은 응답 → 같은 처방 (일관성).
